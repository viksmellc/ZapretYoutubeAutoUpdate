# Zapret Youtube Auto Update

Простая программа для запуска Zapret.

## Как использовать

1. **Запустите программу** (двойной клик)
2. **Разрешите права администратора** при запросе
3. **Программа сделает всё автоматически**

## Что делает программа

- Проверяет и устанавливает необходимые компоненты
- Скачивает последнюю версию Zapret
- Останавливает старые версии
- Запускает новый Zapret

## Особенности

- **Автоматическое обновление** - всегда последняя версия
- **Безопасно** - не трогает системные файлы
- **Просто** - работает в один клик

## Настройки

После первого запуска появится файл `zapret-config.txt`. Его можно открыть и изменить если нужно:

```ini
DefaultBatFile = ваш-файл.bat
```

## Добавление своих сайтов

После запуска появится файл `my-general.txt`. Добавьте в него свои сайты между маркерами:

```txt
# Пользовательские записи
# Добавляйте сайты между маркерами
# USER CUSTOM ENTRIES START
play.google.com
ваш-сайт.com
# USER CUSTOM ENTRIES END
```

## Что делать если не работает

1. Убедитесь, что есть интернет
2. Запустите программу от имени администратора
3. Если ошибка .NET - установите .NET 8

Всё остальное программа сделает сама!
